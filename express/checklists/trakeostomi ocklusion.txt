Title/2. Larma
Text/
Color/<#5EA35C><Indikation:>
• Trakeostomi och A/B problem

Color/<#a5715a><Åtgärd:>
• Luftvägslarm
• ÖNH 71061



Title/3. Lyssna/titta mun + stoma
Text/
Color/<#a5715a><Åtgärd:>
• Bedöm om luften passerar genom mun och/eller stoma



Title/4. Syrgas mun + stoma
Text/
Color/<#a5715a><Åtgärder:>
• 15 L/min via mask över mun
• 15 L/min via mask över stoma



Title/5. Ta bort inre kanyl
Text/
Color/<#5EA35C><Indikation:>
• Alla med inre kanyl och potentiell A/B problem

Color/<#a5715a><Åtgärder:>
• Ta bort inre kanyl
• Spara kanylen, kan behövas för övertrycksventilation



Title/6. För in sugkateter
Text/
Color/<#5EA35C><Indikation:>
• För att bekräfta att yttre kanyl är öppen

Color/<#a5715a><Åtgärd:>
• För in sugkateter genom trak-tuben

Color/<#a5715a><Om du KAN föra in katetern>
• Sug max 5-10 sek i taget
• Ventilera vid behov genom trak-tuben med blåsa
• Kan INTE ventilera: Color/<#A336C9><7. Ta bort yttre kanyl>
• Patient förbättras: Color/<#A336C9><10. Vid förbättring>

Color/<#a5715a><Om du INTE kan föra in katetern>
• Kuffa ur trak-tuben om kuff finns
• Patient blir inte bättre: Color/<#A336C9><7. Ta bort yttre kanyl>
• Patient förbättras: Color/<#A336C9><10. Vid förbättring>



Title/7. Ta bort yttre kanyl
Text/
Color/<#5EA35C><Indikation:>
• Eftersom kanylen kan hamnat fel och obstruera luftvägen

Color/<#a5715a><Åtgärder:>
• Kuffa ur trak-tuben om kuff finns
• Ta bort yttre kanyl



Title/8. Ventilera via mun eller stoma
Text/
Color/<#5EA35C><Indikation:>
• Kvarstående otillräcklig ventilation

Color/<#a5715a><Ventilera via mun:>
• Ockludera stoma med kompresser eller händer
• Två-person mask ventilation över munnen
• Bekräfta ventilation med EtCO2

Color/<#a5715a><Ventilera via stoma:>
• Barnmask eller larynxmask omkring stoma
• Bekräfta ventilation med EtCO2



Title/9. Intubation via stoma
Text/
Color/<#5EA35C><Indikation:>
• Kvarstående otillräcklig ventilation

Color/<#a5715a><Val:>
• ET tub 6 över fiberlaryngoskop
• ET tub 6 över bougie



Title/10. Vid förbättring
Text/
Color/<#5EA35C><Indikation:>
• Patienten blivit bättre, ev partiell obstruktion

Color/<#a5715a><Åtgärder att överväga:>
• Droppa 2-3 ml NaCl ned i trakeostomi för att lösa upp slemmet, sug igen
• Nebulisera 5 ml NaCl via munnen eller stoma för att lösa upp slemmet
